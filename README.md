# Servidor-Minecraft <img src="https://cdn-icons-png.flaticon.com/512/15459/15459593.png" alt="Logo da corrente" style="width: 20px; height: 20px; ">
### Projeto para criar um servidor para jogar Minecraft. 	

De maneira bem resumida, esse projeto possui um codespace dedicado para rodar um ambiente Linux <img src="https://cdn-icons-png.flaticon.com/512/6124/6124995.png" alt="Logo da corrente" style="width: 20px; height: 20px; "> que vai suportar o servidor. Esse servidor ele foi criado com a ajuda de uma tecnologia chamada Crafty Controller <img src="https://cdn-icons-png.flaticon.com/512/10341/10341916.png" alt="Logo da corrente" style="width: 20px; height: 20px; ">, essa ferramenta cria e auxilia com a configuração do servidor, depois ainda faz o trabalho de rodar o servidor de minecraft. Para conseguir atribuir um IP e um domínio que fosse possível acessar, foi utilizado outra tecnologia chamada Playit.gg <img src="https://cdn-icons-png.flaticon.com/512/550/550504.png" alt="Logo da corrente" style="width: 20px; height: 20px; "> que cria um agente para liberar uma conexão para o seu servidor.

Esse projeto não foi criado por mim, nem foram realizados grandes ajustes, eu simplesmente decidi experimentar essa maneira de criar um servidor após ver um tutorial que falava sobre esse assunto no Youtube. 

Agora que o projeto deu certo, tenho uma nova aventura para vivenciar <img src="https://cdn-icons-png.flaticon.com/512/2822/2822374.png" alt="Logo da corrente" style="width: 40px; height: 40px; "> <img src="https://cdn-icons-png.flaticon.com/512/6081/6081840.png" alt="Logo da corrente" style="width: 40px; height: 40px; ">.
